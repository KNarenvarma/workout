# Workout App v6 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rewrite `index.html` as workout tracker v6: weekday schedule with push/miss/skip handling, gym profiles with per-gym PRs, simpler set logging, diet removed, xlsx/JSON/AI-prompt exports, refreshed dark UI.

**Architecture:** Single self-contained `index.html` (no build, no CDN JS, Google Font only). State object `S` persisted to localStorage `train_v6`, migrated from `train_v5`/`grind_v4` on first load. Photos stay in IndexedDB `grind_db` (unchanged). Render = full re-render per tab (v5 pattern, kept). New file assembled from part files in scratchpad concatenated by script; exercise data `W` copied verbatim from v5.

**Tech Stack:** Vanilla JS, inline SVG charts, localStorage + IndexedDB, hand-rolled ZIP/XLSX writer.

## Global Constraints

- Single file `index.html`; no external JS/CSS; only external ref: Google Fonts Inter (with system-font fallback).
- Storage keys: read `train_v6` → fallback `train_v5` → `grind_v4`; **never delete or write old keys**.
- Exercise content (`const W`) copied **verbatim** from v5 — names, sets, reps, rest, muscle, yt, alts, steps, tips.
- `SCHED` weekday mapping unchanged: Mon upper1, Tue lower1, Wed rest, Thu upper2, Fri lower2, Sat/Sun rest.
- Mobile-first, dark theme, Inter, min 44px touch targets, bottom tab bar: Today / Train / Progress / More.
- All diet/meal/breakfast/macro code and data removed.
- Date keys everywhere: `YYYY-MM-DD` local time.

Migration (complete):
```js
function migrateV5(d){
  const out = {version:6, gyms:[{id:"g1",name:"My Gym"}], currentGym:"g1",
    days:{}, overrides:{}, weights:d.weights||[], exNotes:d.notes||{}, photos:d.photos||[]};
  for (const [date, byWo] of Object.entries(d.log||{})){
    const day = out.days[date] = out.days[date] || {sets:{}};
    for (const [wk, byEx] of Object.entries(byWo)){ day.sets[wk] = byEx; }
  }
  for (const date of Object.keys(d.skipped||{})){
    (out.days[date] = out.days[date] || {sets:{}}).status = "skipped";
  }
  return out;   // meals/breakfast intentionally dropped
}
```

XLSX writer (complete algorithm, `parts/13-xlsx.js`):
```js
// CRC32 (table), ZIP with method 0 (stored):
// each entry: local header PK\x03\x04 v20, flags 0x0800(UTF8), method 0,
// dostime/dosdate from fixed 2026 const, crc32, size, size, namelen, 0, name, data
// central dir PK\x01\x02 per entry + EOCD PK\x05\x06.
// XLSX parts: [Content_Types].xml (workbook, worksheets 1..5, rels),
// _rels/.rels -> xl/workbook.xml,
// xl/workbook.xml (5 <sheet name=.. sheetId=N r:id=rIdN/>),
// xl/_rels/workbook.xml.rels (rIdN -> worksheets/sheetN.xml),
// xl/worksheets/sheetN.xml: <sheetData><row r="1"><c r="A1" t="inlineStr"><is><t>..</t></is></c>...
// numbers: <c r=".." ><v>123</v></c>. XML-escape all strings. colRef(i)=A..Z,AA..
// Blob([bytes], {type:"application/vnd.openxmlformats-officedocument.spreadsheetml.sheet"}) → a.download="naren-workout.xlsx"
```
Sheets: `Sessions` (Date, Day, Workout, Status, Gym, SetsDone, Volume kg, Note), `Sets` (Date, Workout, Exercise, Set, Weight, Reps, PR, Gym), `BodyWeight` (Date, Kg), `PRs` (Exercise, Gym, Weight, Reps, Date), `Program` (Workout, Exercise, Sets, Reps, Rest s).
